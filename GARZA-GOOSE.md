# Garza Goose

A pre-configured distribution of [block/goose](https://github.com/block/goose) that connects to a self-hosted `goosed` backend at `https://goosed.garzaos.cloud`.

## One-shot install

### macOS / Linux

```bash
curl -fsSL https://downloads.garzaos.cloud/install.sh | bash
```

### Windows (PowerShell as Admin)

```powershell
iwr -UseB https://downloads.garzaos.cloud/install.ps1 | iex
```

See `scripts/install.sh` and `scripts/install.ps1` for the full installer source.
